# GWAS (PLINK) + GWAS meta-analysis (META v1.7)

This folder contains command-line records used for GWAS association testing and meta-analysis.

## Contents

- `plink_commands.md`  
  PLINK commands used for GWAS QC and association testing.

- `metaGWAS_commands.md`  
  Commands used for GWAS meta-analysis using **META v1.7**.

## Notes

- These files provide the exact command-line workflow used in this study.
- Input genotype data and cohort-level summary statistics are not distributed with this repository.
- Users should prepare their own input files following the required formats of PLINK and META v1.7.

## Software

- PLINK (for GWAS association testing)
- META v1.7 (for GWAS meta-analysis)
